# 診所營運

## 目標
- 將診所營運流程制度化，減少對個別人手與臨場判斷的依賴
- 改善藥房、排班、預約協調與內部工具使用效率
- 為規模化與返英後遠距管理打底

## 人手配置準則（SOP草稿）

### 排更人手標準
| 情況 | 人手要求 |
|------|----------|
| 全日10個起 / 半日6個起（星期六盡量3位） | 一定多過1個人手 |
| 全日7-9個 / 半日6個內 | 可考慮1-2個人手 |
| 全日6-7個以下 / 半日4-5個以下 | 1個人手已夠 |

（詳見下方「醫師與姑娘合作指引 2025」完整版）

## 醫師與姑娘合作指引（非繁忙時段）2025

### 一、醫師協助事項

*治療房及診療區基本管理*
• 熟悉診所物資擺放/電制位置，必要時可自取
• 自行執行：執針灸房、自己執自己嘅藥、fill紙巾、換水
• 清潔治療用品：使用過的拔罐預備下次使用
• 有空餘時間：協助清空治療房垃圾、執針灸房

*治療房準備工作*
• 負責 set 房：預備衫褲/毛巾/床墊、更換新床墊
• 確保治療物品（針灸、拔罐）妥善清晰放置
• 已用治療用品包裝：直接扔入垃圾桶，勿隨處棄置

*自主帶領病人*
• 自己帶病人入治療房（女性病人換衣時請姑娘協助）
• 男性醫師特別注意女性病人隱私
• 修改藥方時用內線電話通知姑娘（避免遺漏）
• 平時透過 WhatsApp 提前通知藥方變更

### 二、溝通機制
• 緊急需求：WhatsApp 冇覆 → 用內線電話
• 尊重姑娘服務崗位，保持友善態度
• 醫師應有義務了解物品存放位置

### 三、前台與藥房區分工

*姑娘負責（前台+藥房）*
• 開診前準備：醫師袍、開電腦、檢查日用品
• 前台：客服查詢、預約、掛號、收錢、分單、處理到診紙/病假紙
• 藥房：執藥、配藥、換藥+扣藥表、預熱外敷藥、寄藥事項

*醫師負責（醫師房+治療房）*
• 叫病人名入房看診
• 預備所需物品、帶病人入房、進行治療
• 了解物資存放位置，切勿長期過度依賴姑娘

### 四、特殊情況處理

*服藥時間說明*
• 下午就診但藥方寫「早上/中午」→ 醫師須向病人說明今日服藥安排
• 減少病人回頭詢問，減輕姑娘負擔

*新症/英語病人沖藥指引*
• 醫師在「中醫在線」設置快捷鍵（服藥方法）
• 標準指引：80-100ml 熱水 → 放藥粉 → 泡焗3-5分鐘 → 攪拌待溫才飲

*統一藥丸用量格式*
• 每天服__次，每次__粒 + 特別指引
• 避免醫師誤打（如 1g ≠ 1粒）

*孕婦特殊處理*
• 出藥方同時必須先通知姑娘
• 姑娘用新藥匙配藥確保安全

### 預計需要處理嘅爭議點
- 其他病人重配藥：幾多包可即日取 vs 幾多包翌日取？
- 病人跨分店取藥 / 寄藥嘅處理流程
